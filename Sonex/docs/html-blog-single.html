Comments

This section show user's comments and must be wrapped to .comments.

Each comment has next layout: photo, name, time and message.

comment layout
<div class="comment">

		<div class="col-md-8">
			<div class="user-photo">
				<img src="img/blog/comment-1.png" alt="">
			</div>
